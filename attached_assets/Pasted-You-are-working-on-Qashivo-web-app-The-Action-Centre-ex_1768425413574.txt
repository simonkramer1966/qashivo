PRIMARY GOAL

ABSOLUTE DESIGN RULES (must follow)
1) Remove cards wherever possible

- Prefer weight + spacing, not color, to signal importance.

- Default UI is neutral/monochrome.
- Accent color only for primary action (Approve) and selection state.
- Muted semantic tints only for exceptions (Attention, Dispute, Query, Broken).

5) Layout: calm, aligned, breathable
- Increase whitespace: larger top/bottom padding, clear spacing between sections.
- Sticky tab bar and sticky table headers where applicable.
- Lists should resemble Apple Wallet / iCloud tables: clean rows, subtle hover, clear columns.
